requires Unity 2019.3.0+

IL.ranch, 2020. 
ILonion32@gmail.com


This package contains HDRP and LWRP (Universal) assets.
In order to enable prefabs you should enable HDRP or LWRP pipeline.
Step by step:
1. Open package manager;
2. Install HDRP or LWRP (Universal) pipeline;
3. Open project settings;
4. Assign HDRP or LWRP asset from 'PipAsset' folder into Graphics > Scriptable Render Pipeline Settings;
Done. Prefabs ready to use.
